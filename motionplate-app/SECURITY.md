## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately before disclosing it publicly.

### How to Report

- Send an email to: security@motionplate.dev
- Include detailed information about the vulnerability
- Provide steps to reproduce if possible
- Include your contact information for follow-up

### Response Process

1. We will acknowledge receipt within 48 hours
2. We will provide a detailed response within 7 days
3. We will fix the vulnerability in a timely manner
4. We will coordinate disclosure with you

### Security Best Practices

- Keep dependencies updated
- Use security scanning tools
- Follow secure coding practices
- Review code for common vulnerabilities

## Security Scanning

This project uses automated security scanning:
- Snyk for dependency vulnerability scanning
- Code analysis for security issues
- CI/CD pipeline security checks
